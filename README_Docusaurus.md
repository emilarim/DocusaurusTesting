To bring the Docusaurus Website configuration from GitHub to your laptop and run it locally, please follow these instructions:

1. Before downloading the project, verify these install required tools.

- Node.js
- Git
- A code editor like Visual Studio Code

   ```bash
   node -v
   npm -v
   git --version

   ```


2. Go to the folder where you want the poject.

   Example:

   ```bash
   cd Documents\Generators\Docusaurus
   ```

3. Clone the repository:

   ```bash
   git clone https://github.com/emilarim/DocusaurusTesting.git
   ```

4. Check if the folder was downloaded in the proposal folder.

5. Install dependencies.

   ```bash
   npm install
   ```
6. Run the local development server

   ```bash
   npm run start
   ```

This command starts a local development server and opens up a browser window. 

`http://localhost:3000/DocusaurusTesting/`
